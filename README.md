## 📘 About The Project
Tech Blog Website is a full-stack MERN application where developers and tech enthusiasts can create, manage, and share blog posts.

It provides a modern UI, authentication system, and a powerful dashboard for managing content.

- 📊 Personal Dashboard
- 📄 Blog Details Page
